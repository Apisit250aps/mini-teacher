# นโยบายความเป็นส่วนตัว (Privacy Policy)

**เวอร์ชัน:** 1.0.0  
**วันที่มีผลบังคับใช้:** 12 มีนาคม 2026  
**ปรับปรุงล่าสุด:** 12 มีนาคม 2026

## บทนำ

**Mini Teacher** ("เรา", "ผู้ให้บริการ") ให้ความสำคัญสูงสุดกับความเป็นส่วนตัวของคุณ นโยบายฉบับนี้อธิบายอย่างชัดเจนว่าเราเก็บรวบรวม ใช้ เปิดเผย และปกป้องข้อมูลส่วนบุคคลของคุณอย่างไรเมื่อคุณใช้บริการของเรา

นโยบายนี้จัดทำขึ้นเพื่อให้เป็นไปตามพระราชบัญญัติคุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562 (PDPA — Personal Data Protection Act)

**การใช้งานบริการของเราถือว่าคุณยอมรับนโยบายความเป็นส่วนตัวฉบับนี้**

## 1. ข้อมูลที่เราเก็บรวบรวม

### 1.1 ข้อมูลที่คุณให้โดยตรง

เมื่อลงทะเบียนหรือใช้งานระบบ เราเก็บข้อมูลต่อไปนี้:

| หมวดข้อมูล | รายละเอียด | วัตถุประสงค์ |
|---|---|---|
| **ข้อมูลบัญชีผู้ใช้** | ชื่อ-นามสกุล, อีเมล, รหัสผ่าน (เข้ารหัส) | ระบุตัวตนและเข้าสู่ระบบ |
| **ข้อมูลโปรไฟล์** | ชื่อแสดง, รูปโปรไฟล์ | แสดงในหน้าโปรไฟล์ |
| **ข้อมูลชั้นเรียน** | ปีการศึกษา, เทอม, ชื่อวิชา, คำอธิบาย | จัดการชั้นเรียน |
| **ข้อมูลนักเรียน** | ชื่อนักเรียน, รหัสนักเรียน | บันทึกรายชื่อและผล |
| **ข้อมูลการเช็คชื่อ** | วันที่เช็คชื่อ, สถานะการมาเรียน | บันทึกการเข้าเรียน |
| **ข้อมูลคะแนน** | ชื่องาน, ประเภท, คะแนนที่ได้ | ประเมินผลการเรียน |

### 1.2 ข้อมูลที่ระบบเก็บโดยอัตโนมัติ

เมื่อคุณใช้งานบริการ ระบบจะบันทึกข้อมูลต่อไปนี้โดยอัตโนมัติ:

- **IP Address** — เพื่อความปลอดภัยและป้องกันการเข้าถึงโดยไม่ได้รับอนุญาต
- **User Agent** — ข้อมูลเบราว์เซอร์และอุปกรณ์ที่ใช้ (เช่น Chrome on Windows)
- **เวลาการยอมรับเอกสาร** — วันและเวลาที่คุณยอมรับ นโยบายนี้และข้อกำหนดการใช้งาน
- **Session Token** — โทเค็นสำหรับรักษาสถานะการเข้าสู่ระบบ

### 1.3 ข้อมูลจากบุคคลที่สาม (Third Party)

หากคุณเลือกเข้าสู่ระบบผ่านบริการของบุคคลที่สาม (เช่น Google OAuth) เราจะได้รับ:

- ชื่อบัญชีและอีเมลจากผู้ให้บริการนั้น
- ข้อมูลโปรไฟล์พื้นฐานตามที่คุณอนุญาต

---

## 2. วิธีที่เราใช้ข้อมูล

เราใช้ข้อมูลที่เก็บรวบรวมเพื่อวัตถุประสงค์ดังต่อไปนี้:

### 2.1 การให้บริการหลัก
- **ยืนยันตัวตน** — เข้าสู่ระบบและรักษาสถานะการใช้งาน
- **จัดการชั้นเรียน** — แสดงผล ค้นหา และบันทึกข้อมูลการเรียนการสอน
- **บันทึกพฤติกรรมการเรียน** — ติดตามการมาเรียนและผลการเรียนของนักเรียน

### 2.2 ความปลอดภัย
- ตรวจจับและป้องกันการเข้าถึงโดยไม่ได้รับอนุญาต
- บันทึก Log การยืนยันการรับทราบนโยบาย (IP, User Agent, เวลา) เพื่อใช้เป็นหลักฐาน

### 2.3 การปรับปรุงบริการ
- วิเคราะห์การใช้งานโดยรวม (แบบไม่ระบุตัวตน) เพื่อพัฒนาประสบการณ์ผู้ใช้

---

## 3. ฐานทางกฎหมายในการประมวลผลข้อมูล (Legal Basis)

ภายใต้ PDPA เราอ้างอิงฐานทางกฎหมายต่อไปนี้:

| ฐานทางกฎหมาย | การใช้งาน |
|---|---|
| **ความยินยอม (Consent)** | การเก็บข้อมูล IP/User Agent เพื่อบันทึกการยอมรับนโยบาย |
| **การปฏิบัติตามสัญญา (Contract)** | ข้อมูลบัญชีและชั้นเรียนที่จำเป็นสำหรับการให้บริการ |
| **ประโยชน์โดยชอบธรรม (Legitimate Interest)** | ข้อมูล Session และความปลอดภัยของระบบ |

---

## 4. การแบ่งปันข้อมูลกับบุคคลที่สาม

### 4.1 เราไม่ขายข้อมูลของคุณ

เราไม่ขาย ให้เช่า หรือถ่ายโอนข้อมูลส่วนบุคคลของคุณเพื่อวัตถุประสงค์เชิงพาณิชย์ไม่ว่าในกรณีใด

### 4.2 ผู้ให้บริการภายนอกที่เราใช้

เราใช้บริการต่อไปนี้เพื่อการดำเนินงาน:

| ผู้ให้บริการ | วัตถุประสงค์ | นโยบายความเป็นส่วนตัว |
|---|---|---|
| **Vercel** | Hosting และ Analytics | [vercel.com/legal/privacy-policy](https://vercel.com/legal/privacy-policy) |
| **PostgreSQL** | ฐานข้อมูลระบบ | ดำเนินการภายใน |
| **NextAuth.js** | ระบบยืนยันตัวตน | [authjs.dev](https://authjs.dev) |
| **Google OAuth** (ถ้ามี) | เข้าสู่ระบบผ่าน Google | [policies.google.com/privacy](https://policies.google.com/privacy) |

ผู้ให้บริการภายนอกเหล่านี้เข้าถึงข้อมูลเฉพาะที่จำเป็นสำหรับหน้าที่ของตน และผูกพันด้วยข้อตกลงการประมวลผลข้อมูลที่เหมาะสม

### 4.3 เหตุผลทางกฎหมาย

เราอาจเปิดเผยข้อมูลหากจำเป็นต้องปฏิบัติตามคำสั่งศาล กฎหมาย หรือกระบวนการทางกฎหมาย

## 5. การเก็บรักษาข้อมูล (Data Retention)

| ประเภทข้อมูล | ระยะเวลาเก็บ | เหตุผล |
|---|---|---|
| ข้อมูลบัญชีผู้ใช้ | จนกว่าจะลบบัญชี + 30 วัน | สำรองข้อมูลก่อนลบถาวร |
| ข้อมูลชั้นเรียนและนักเรียน | จนกว่าเจ้าของจะลบ | ข้อมูลเป็นของผู้ใช้งาน |
| บันทึกการยอมรับนโยบาย (UserAcceptance) | 3 ปี | หลักฐานทางกฎหมายตาม PDPA |
| Session Token | ตามวันหมดอายุของ Session | ความปลอดภัยของระบบ |
| IP Address / User Agent | 3 ปี | หลักฐานการยินยอม |

## 6. ความปลอดภัยของข้อมูล

เราใช้มาตรการต่อไปนี้เพื่อปกป้องข้อมูลของคุณ:

- **การเข้ารหัสรหัสผ่าน** — ใช้ Argon2 ซึ่งเป็นอัลกอริทึมการแฮชที่แนะนำโดย OWASP
- **การส่งข้อมูลผ่าน HTTPS** — ข้อมูลทั้งหมดส่งผ่านการเชื่อมต่อที่เข้ารหัส
- **การตรวจสอบสิทธิ์** — ทุก API Endpoint ต้องผ่านการยืนยันตัวตน
- **การควบคุมสิทธิ์เข้าถึง** — ผู้ใช้งานเข้าถึงได้เฉพาะข้อมูลของตนเอง
- **ฐานข้อมูลที่ปลอดภัย** — PostgreSQL พร้อมการกำหนดสิทธิ์ที่เหมาะสม

> **หมายเหตุ:** แม้เราจะใช้มาตรการรักษาความปลอดภัยที่ดีที่สุด แต่ไม่มีระบบใดที่ปลอดภัย 100% เราขอแนะนำให้คุณเลือกรหัสผ่านที่แข็งแรงและไม่แบ่งปันข้อมูลการเข้าสู่ระบบ

## 7. สิทธิ์ของเจ้าของข้อมูล

ภายใต้ PDPA คุณมีสิทธิ์ดังต่อไปนี้:

### 7.1 สิทธิ์ที่คุณมี

| สิทธิ์ | รายละเอียด |
|---|---|
| **สิทธิ์เข้าถึง (Access)** | ขอดูข้อมูลส่วนบุคคลที่เราเก็บไว้ |
| **สิทธิ์แก้ไข (Rectification)** | ขอแก้ไขข้อมูลที่ไม่ถูกต้อง |
| **สิทธิ์ลบ (Erasure)** | ขอลบข้อมูลของคุณ ("สิทธิ์ที่จะถูกลืม") |
| **สิทธิ์จำกัดการใช้ (Restriction)** | ขอระงับการประมวลผลข้อมูลชั่วคราว |
| **สิทธิ์โอนย้ายข้อมูล (Portability)** | ขอรับข้อมูลของคุณในรูปแบบที่ใช้งานได้ |
| **สิทธิ์คัดค้าน (Objection)** | คัดค้านการประมวลผลข้อมูลในบางกรณี |
| **สิทธิ์ถอนความยินยอม (Withdraw Consent)** | ถอนความยินยอมที่เคยให้ไว้ได้ตลอดเวลา |

### 7.2 วิธีใช้สิทธิ์

คุณสามารถใช้สิทธิ์เหล่านี้ได้โดยติดต่อเราผ่านช่องทางที่ระบุไว้ในหัวข้อ "การติดต่อ" เราจะดำเนินการตอบสนองภายใน **30 วัน** นับจากวันที่ได้รับคำขอ

## 8. คุกกี้และเทคโนโลยีติดตาม

### 8.1 คุกกี้ที่เราใช้

| ประเภท | ชื่อ | วัตถุประสงค์ |
|---|---|---|
| **จำเป็น (Essential)** | `next-auth.session-token` | รักษาสถานะการเข้าสู่ระบบ |
| **จำเป็น (Essential)** | `next-auth.csrf-token` | ป้องกัน CSRF Attack |
| **Analytics** | Vercel Analytics | วิเคราะห์การใช้งานในภาพรวม (ไม่ระบุตัวตน) |

### 8.2 การจัดการคุกกี้

คุณสามารถปิดการใช้งานคุกกี้ผ่านการตั้งค่าเบราว์เซอร์ของคุณ อย่างไรก็ตาม การปิดคุกกี้ที่จำเป็นอาจทำให้ไม่สามารถเข้าสู่ระบบได้

## 9. เด็กและเยาวชน

บริการนี้ไม่ได้ออกแบบมาสำหรับเด็กและเยาวชนที่อายุต่ำกว่า **18 ปี** ในฐานะผู้ใช้งาน ผู้ใช้งานมีหน้าที่รับผิดชอบในการจัดการข้อมูลนักเรียนที่เป็นผู้เยาว์ตาม PDPA และกฎหมายที่เกี่ยวข้อง โดยต้องได้รับความยินยอมจากผู้ปกครองหรือผู้มีอำนาจตามกฎหมายก่อนบันทึกข้อมูลเด็ก

---

## 10. การถ่ายโอนข้อมูลระหว่างประเทศ

บริการนี้โฮสต์บน Vercel ซึ่งอาจมีการจัดเก็บข้อมูลในเซิร์ฟเวอร์ต่างประเทศ เราตรวจสอบให้แน่ใจว่าผู้ให้บริการเหล่านี้มีมาตรการคุ้มครองข้อมูลที่เหมาะสมตามมาตรฐานสากล

---

## 11. การเปลี่ยนแปลงนโยบาย

- เราอาจปรับปรุงนโยบายนี้เป็นครั้งคราวเพื่อให้สอดคล้องกับการเปลี่ยนแปลงของบริการหรือกฎหมาย
- เมื่อมีการเปลี่ยนแปลงสำคัญ เราจะแจ้งให้ทราบผ่านหน้าระบบหรืออีเมลล่วงหน้า **อย่างน้อย 30 วัน**
- เวอร์ชันและวันที่จะแสดงไว้ที่ด้านบนของเอกสารนี้เสมอ
- การใช้งานบริการต่อเนื่องหลังจากนโยบายมีผลถือว่าคุณยอมรับนโยบายฉบับใหม่
