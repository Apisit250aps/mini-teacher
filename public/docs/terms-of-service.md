## บทนำ

ยินดีต้อนรับสู่ **Mini Teacher** — แพลตฟอร์มจัดการชั้นเรียนออนไลน์สำหรับคุณครูและบุคลากรทางการศึกษา ที่ช่วยให้การเช็คชื่อ บันทึกคะแนน และติดตามพัฒนาการนักเรียนเป็นเรื่องง่ายในที่เดียว

ข้อกำหนดการใช้งานฉบับนี้ ("ข้อตกลง") เป็นสัญญาที่มีผลผูกพันทางกฎหมายระหว่างคุณ ("ผู้ใช้งาน") กับผู้พัฒนาระบบ Mini Teacher ("ผู้ให้บริการ") กรุณาอ่านข้อกำหนดเหล่านี้อย่างละเอียดก่อนเริ่มใช้งานบริการ

**การเข้าถึงหรือใช้งาน Mini Teacher ถือว่าคุณยอมรับข้อกำหนดเหล่านี้ทั้งหมด** หากคุณไม่ยอมรับข้อกำหนดใดๆ โปรดหยุดใช้งานบริการทันที

## 1. คำนิยาม

| คำ | ความหมาย |
|---|---|
| **บริการ** | เว็บแอปพลิเคชัน Mini Teacher ที่ให้บริการผ่าน miniteacher.online และ API ที่เกี่ยวข้องทั้งหมด |
| **ผู้ใช้งาน** | บุคคลหรือองค์กรที่ลงทะเบียนและใช้งานบริการ ได้แก่ ผู้ดูแลระบบ ครูผู้สอน |
| **เนื้อหา** | ข้อมูล ข้อความ รูปภาพ และสื่อต่างๆ ที่ผู้ใช้งานจัดเก็บหรือส่งผ่านระบบ |
| **ข้อมูลชั้นเรียน** | ข้อมูลปีการศึกษา เทอม ชั้นเรียน รายชื่อนักเรียน ข้อมูลการเช็คชื่อ คะแนน และงานที่มอบหมาย |
| **บัญชีผู้ใช้** | ข้อมูลที่ใช้ระบุตัวตนในระบบ ประกอบด้วยชื่อ อีเมล และรหัสผ่านที่เข้ารหัส |

## 2. การลงทะเบียนและบัญชีผู้ใช้

### 2.1 คุณสมบัติผู้ใช้งาน

- คุณต้องมีอายุไม่น้อยกว่า **18 ปีบริบูรณ์** หรือมีความสามารถตาม กฎหมายในการทำสัญญาที่มีผลผูกพัน
- คุณต้องให้ข้อมูลที่ถูกต้อง ครบถ้วน และเป็นปัจจุบันในการลงทะเบียน
- 1 บุคคลสามารถลงทะเบียนได้ **1 บัญชีผู้ใช้** เท่านั้น

### 2.2 ความรับผิดชอบด้านบัญชี

- คุณมีหน้าที่รักษาความปลอดภัยของรหัสผ่านและข้อมูลการเข้าสู่ระบบ
- ห้ามแบ่งปันบัญชีผู้ใช้ให้บุคคลอื่น
- คุณต้องแจ้งผู้ให้บริการทันทีหากพบว่าบัญชีถูกใช้งานโดยไม่ได้รับอนุญาต
- กิจกรรมทั้งหมดที่เกิดขึ้นในบัญชีของคุณเป็นความรับผิดชอบของคุณ

### 2.3 ระดับสิทธิ์การใช้งาน

ระบบมี 3 ระดับสิทธิ์:

| ระดับ | สิทธิ์ |
|---|---|
| **ผู้ดูแลระบบ (Admin)** | จัดการผู้ใช้งาน เอกสารระบบ และการตั้งค่าทั้งหมด |
| **ครูผู้สอน (Teacher)** | สร้างและจัดการชั้นเรียน เช็คชื่อ บันทึกคะแนน |
| **ผู้ใช้ทั่วไป** | เข้าถึงข้อมูลตามที่ได้รับอนุญาต |

## 3. การใช้งานที่ได้รับอนุญาต

### 3.1 สิ่งที่ทำได้

คุณได้รับอนุญาตให้ใช้บริการเพื่อ:

- **จัดการชั้นเรียน** — สร้างปีการศึกษา เทอม และรายการวิชา
- **บันทึกรายชื่อนักเรียน** — เพิ่ม แก้ไข และลบข้อมูลนักเรียน
- **เช็คชื่อเข้าเรียน** — บันทึกสถานะการมาเรียน (มาเรียน/ขาด/มาสาย/ลา)
- **บันทึกคะแนน** — จัดการงานที่มอบหมาย แบบทดสอบ และคะแนนรวม
- **ดูรายงาน** — วิเคราะห์ข้อมูลการเข้าเรียนและผลการเรียน

### 3.2 สิ่งที่ห้ามทำ

ผู้ใช้งาน **ห้าม** กระทำสิ่งต่อไปนี้โดยเด็ดขาด:

- ใช้บริการเพื่อวัตถุประสงค์ที่ผิดกฎหมายหรือก่อความเสียหาย
- พยายามเข้าถึง แฮก หรือรบกวนระบบโดยไม่ได้รับอนุญาต
- บันทึกข้อมูลเท็จหรือทำให้ข้อมูลนักเรียนผิดเพี้ยน
- รวบรวม ขาย หรือถ่ายโอนข้อมูลของผู้อื่นโดยไม่ได้รับความยินยอม
- ใช้บอทหรือเครื่องมืออัตโนมัติเพื่อสแกนหรือดึงข้อมูลจากระบบ
- ทำซ้ำ แจกจ่าย หรือดัดแปลงซอฟต์แวร์โดยไม่ได้รับอนุญาต
- ใช้บริการเพื่อส่งสแปมหรือเนื้อหาที่ไม่เหมาะสม

## 4. ความเป็นเจ้าของข้อมูล

### 4.1 ข้อมูลของผู้ใช้งาน

- **คุณยังคงเป็นเจ้าของข้อมูลชั้นเรียน ข้อมูลนักเรียน และข้อมูลการเรียนการสอนทั้งหมด** ที่คุณป้อนเข้าสู่ระบบ
- ผู้ให้บริการจะไม่อ้างสิทธิ์ความเป็นเจ้าของในข้อมูลดังกล่าว
- คุณมีสิทธิ์ส่งออกหรือลบข้อมูลของคุณได้ตลอดเวลา (ภายในขอบเขตที่ระบบรองรับ)

### 4.2 ทรัพย์สินทางปัญญา

- ซอฟต์แวร์ การออกแบบ UI โลโก้ และเนื้อหาที่ผู้ให้บริการสร้างขึ้นทั้งหมดเป็นทรัพย์สินของผู้ให้บริการ
- ผู้ใช้งานได้รับสิทธิ์ใช้งาน (License) แบบไม่ผูกขาด ไม่สามารถโอนสิทธิ์ได้ เพื่อใช้บริการตามวัตถุประสงค์ที่ระบุ

## 5. ความเป็นส่วนตัวและการคุ้มครองข้อมูล

- การเก็บและใช้ข้อมูลส่วนบุคคลเป็นไปตาม **นโยบายความเป็นส่วนตัว** (Privacy Policy) ที่แนบมาพร้อมนี้
- ผู้ให้บริการปฏิบัติตามพระราชบัญญัติคุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562 (PDPA)
- ข้อมูลนักเรียนถือเป็นข้อมูลที่มีความอ่อนไหว — ผู้ใช้งานมีหน้าที่รับผิดชอบในการจัดการและปกป้องข้อมูลเหล่านี้ตามกฎหมาย

## 6. การระงับและยุติบัญชี

### 6.1 โดยผู้ให้บริการ

ผู้ให้บริการสงวนสิทธิ์ระงับหรือยุติบัญชีในกรณีต่อไปนี้โดยไม่ต้องแจ้งล่วงหน้า:

- ละเมิดข้อกำหนดการใช้งานฉบับนี้
- กิจกรรมฉ้อโกงหรือผิดกฎหมาย
- ข้อมูลการลงทะเบียนเป็นเท็จ
- ทำให้ระบบหรือผู้ใช้งานอื่นเสียหาย

### 6.2 โดยผู้ใช้งาน

- คุณสามารถยุติบัญชีได้ตลอดเวลาโดยติดต่อผู้ดูแลระบบ
- เมื่อยุติบัญชี ข้อมูลของคุณจะถูกลบออกจากระบบตามนโยบายความเป็นส่วนตัว

## 7. ข้อจำกัดความรับผิด

### 7.1 การให้บริการ "ตามสภาพ"

บริการนี้จัดให้ "ตามสภาพที่เป็น" (As-is) และ "ตามที่มีให้" (As-available) โดยไม่มีการรับประกันใดๆ ไม่ว่าจะโดยชัดแจ้งหรือโดยปริยาย รวมถึงแต่ไม่จำกัดเพียง:

- การรับประกันความเหมาะสมสำหรับวัตถุประสงค์เฉพาะ
- ความถูกต้องสมบูรณ์ของข้อมูล
- ความพร้อมใช้งานอย่างต่อเนื่องไม่มีการหยุดชะงัก

### 7.2 ขอบเขตความรับผิดสูงสุด

ผู้ให้บริการจะไม่รับผิดชอบต่อความเสียหายทางอ้อม ความเสียหายเพิ่มเติม หรือความเสียหายพิเศษใดๆ ที่เกิดจากการใช้หรือไม่สามารถใช้บริการได้

- ผู้ให้บริการสงวนสิทธิ์แก้ไขข้อกำหนดเหล่านี้ได้ตลอดเวลา
- เมื่อมีการเปลี่ยนแปลง จะประกาศแจ้งในระบบหรือผ่านช่องทางการสื่อสารที่ระบุไว้
- การใช้งานบริการต่อเนื่องหลังจากมีการประกาศเปลี่ยนแปลงถือว่าคุณยอมรับข้อกำหนดฉบับใหม่
